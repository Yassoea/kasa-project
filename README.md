# KASA - Projet OpenClassrooms

Bienvenue dans le projet **KASA**, développé dans le cadre de ma formation OpenClassrooms. Ce projet consiste à développer une plateforme de location d'appartements entre particuliers, avec une expérience utilisateur fluide et des fonctionnalités modernes.

## Aperçu

KASA est une application React permettant aux utilisateurs de parcourir et de consulter des annonces de logements. Le projet met en œuvre des bonnes pratiques de développement front-end, telles que la réutilisation des composants, la gestion des états et la création d’une interface responsive.

---

## Technologies utilisées

- **React** : Framework JavaScript pour le développement de l'interface utilisateur.
- **React Router** : Gestion des routes de l'application.
- **JavaScript ()**
- **Node.js et npm** : Gestion des dépendances et scripts.

## Installation

### Prérequis

Assurez-vous d'avoir les éléments suivants installés sur votre machine :

- [Node.js](https://nodejs.org/) (version 14 ou supérieure)
- [npm](https://www.npmjs.com/) (installé avec Node.js)

### Étapes

1. **Cloner le dépôt GitHub :**
   ```bash
   git clone https://github.com/<ton-utilisateur-github>/kasa.git
   cd kasa
2. **Installer les dépendances :**
    ```bash
	npm install
3. **Lancer l'application en mode développement :**
 	```bash
	npm start
**L'application sera accessible à l'adresse :**  
- http://localhost:3000.
